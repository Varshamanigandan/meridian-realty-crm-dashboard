# Meridian Realty Group — CRM Dashboard

## Task
Design a CRM / Client Management Dashboard UI for a small agency or B2B service-based business.

## Business
A small real estate agency needing to track leads, manage clients, and follow up on tasks across a team of agents.

## Prototype
🔗 [View the Figma prototype](PASTE_YOUR_LINK_HERE)

## Screens
- **Dashboard** — Overview of active leads, active clients, deals closed, and pending tasks
- **Lead Pipeline** — Kanban board across 5 stages: New Lead, Contacted, Viewing Scheduled, Offer Made, Closed
- **Client List** — Table view with contact info, status, assigned agent, and budget range
- **Client Profile** — Contact info, property interests, viewing history, agent notes, and communication timeline
- **Task Tracker** — Filterable follow-up list with client, due date, priority, and completion status

## Tools Used
Figma (AI-assisted UI generation + interactive prototyping)

## Key Workflow & Usability Decisions
- Kanban pipeline reflects the real stage-based nature of real estate sales
- Client profile consolidates contact, interests, viewing history, and notes in one place
- Tasks always show their linked client, avoiding lost context
- Color-coded status badges (active/closed/prospect, priority levels) for fast visual triage
